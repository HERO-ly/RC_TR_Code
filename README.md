# RC_TR_Code
 2019-2020 TR机器人源代码

**主要功能:**

	1.	使用到状态机和看门狗.
	2.	用4个3508电机结合位置环控制方向, 4个3508结合速度环控制方向.
	3.	通过左右标定确定方向电机最大最小值.
	4.	通过通信读取Y,X,W三轴目标速度, 实现平移或者原地自转操作.
	5.	控制DS3115舵机转向.
	6.	通过多次位置环操控, 实现踢球.
**用到外设:**

	1.	TIM3 -- 1ms定时器, 控制状态机函数执行间隔
	2.	DMA1与USART2 -- 和上位机通信
	3.	TIM14 -- LED
